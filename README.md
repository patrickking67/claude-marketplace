# DivergeIT — Claude Plugin Marketplace

Plugin marketplace maintained by Patrick King.

## Plugins

| Plugin | What it does |
|---|---|
| [`Timekeeper`](./timekeeper) | Microsoft 365-native legal time reconstruction & billing for Jalmar Properties, Inc. — mines Outlook / Teams / calendar via Claude's native M365 connector, drafts court-defensible entries to the firm's standards, and exports a Clio import CSV plus a polished billing workbook. Skills + agents only — no bundled MCP. |

## Install — Claude Code

    /plugin marketplace add patrickking67/claude-marketplace
    /plugin install timekeeper@DivergeIT

## Install — Cowork / Claude.ai

Add this repository as a marketplace, then install **Timekeeper**. The only requirement on any surface is the Claude-native **Microsoft 365** connector (read) enabled in your workspace. Optional: **Zoom**, **Webex**, **Calendly** for richer meeting signal.
