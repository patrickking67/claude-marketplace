# DivergeIT — Claude Plugin Marketplace

Plugin marketplace by Patrick King.

## Plugins

| Plugin | What it does |
|---|---|
| [`Timekeeper`](./timekeeper) | Microsoft 365-native legal time reconstruction & billing for Jalmar Properties, Inc. — mines Outlook (Sent / Inbox / Deleted) / Teams / calendar / SharePoint via Claude's native M365 connector (plus optional Zoom / Webex / Calendly), drafts court-defensible entries to the firm's standards (§ 330, no-block, per-case policies), and writes a Clio bulk-import CSV plus a polished `.xlsx` workbook to a folder you pick. One-click jump to Clio for one-off entries. Skills + agents only — no bundled MCP. |

## Install — Claude Code

    /plugin marketplace add patrickking67/claude-marketplace
    /plugin install timekeeper@DivergeIT

## Install — Cowork / Claude.ai

Add this repository as a marketplace, then install **Timekeeper**. The only requirement on any surface is the Claude-native **Microsoft 365** connector (read) enabled in your workspace. Optional: **Zoom**, **Webex**, **Calendly** for richer meeting signal.
